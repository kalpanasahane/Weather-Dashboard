#  Real-Time Weather Dashboard in Power BI

An end-to-end Power BI project that leverages a Weather API to deliver real-time and forecast weather insights through an interactive dashboard.

##  Features

* Integrated live weather data using a Weather API and Power BI Web connector.
* Imported and transformed JSON data with Power Query.
* Combined multiple city reports into a unified master table.
* Performed data cleaning, date formatting, and duplicate handling.
* Designed a star schema data model with optimized relationships.
* Created dedicated DAX measures for better model organization.
* Developed an interactive dark-themed dashboard with custom backgrounds.
* Built dynamic KPI cards for temperature, wind speed, pressure, and humidity.
* Added city-based slicers for seamless user interaction.
* Displayed weather conditions using dynamic image URLs and icons.
* Implemented advanced DAX calculations for trend analysis.
* Designed custom circular indicators to visualize air quality (PM10).
* Included status indicators with color-coded thresholds for enhanced readability.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Weather API
* JSON
* Data Modeling (Star Schema)

## Project Highlights

This project demonstrates API integration, data transformation, dimensional modeling, DAX calculations, and interactive dashboard design, making it an excellent portfolio project for aspiring data analysts and business intelligence professionals.
